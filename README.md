# calorie-tracker-telegram

## 商业级前端 Demo（无后端）

当前仓库提供一个可直接演示的前端 MVP：`demo.html`。

### 快速启动
- 方式 1：直接双击 `demo.html` 打开。
- 方式 2：启动本地静态服务：
  - `python3 -m http.server 8000`
  - 浏览器访问 `http://localhost:8000/demo.html`

### Demo 能力（纯前端）
- 专业演示布局：左侧商业说明 + 右侧移动端真机框。
- 核心流程：记录页 → 上传页 → 识别中弹窗 → 结果页 → 分析页 → 我的页。
- 关键交互：
  - 餐次切换 + 餐食描述输入
  - 模拟识别结果与风险标签
  - 换建议（策略池）
  - 保存记录后回写首页 KPI
- 数据能力：
  - `localStorage` 本地持久化记录
  - 近 7 日趋势图动态渲染
  - 可编辑用户档案（目标/身高/体重/活动）

> 说明：当前仅演示前端体验，不包含真实后端识别、鉴权与数据库。
