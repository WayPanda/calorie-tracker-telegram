# calorie-tracker-telegram

## 快速预览（纯前端 Demo）

本仓库提供一个无需后端的静态演示页：`demo.html`。

### 本地打开方式
- 直接双击 `demo.html`；或
- 使用本地静态服务：
  - `python3 -m http.server 8000`
  - 浏览器访问 `http://localhost:8000/demo.html`

### Demo 覆盖流程
- 记录页（首页）
- 上传页（模拟图片选择）
- 结果页（模拟热量识别与建议）
- 分析页（7日趋势）
- 我的页（目标与订阅入口）

> 说明：当前仅为前端交互演示，不包含后端识别与真实数据接口。
